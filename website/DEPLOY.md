# Blueprint AI System — Website Deployment Guide

## Recommended: Deploy on Vercel (Free)
1. Go to https://vercel.com
2. Sign up with GitHub
3. Click **Add New → Project**
4. Import the `website` folder from this directory
5. Project name: `blueprintaisystems`
6. Deploy — your site is live at **`blueprintaisystems.vercel.app`** instantly

## Alternative: GitHub Pages (Free)
1. Create a GitHub repo called `blueprint-ai-website`
2. Upload `index.html` and `styles.css`
3. Go to Settings → Pages → Deploy from `main` branch, `/` root
4. Live at `yourusername.github.io/blueprint-ai-website`

## Alternative: Netlify (Free)
1. Go to https://netlify.com
2. Drag the `website` folder onto the browser window
3. Gets a random URL instantly
